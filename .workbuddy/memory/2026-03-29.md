# 2026-03-29 工作日志

## 11:35 — 修复注册报错 `Duplicate entry '' for key 'uk_nickname'`

**问题**：用户注册时抛出 PDOException，`nickname_lower` 唯一索引冲突。

**根本原因**：`register.php` 的 INSERT 语句字段列表中缺少 `nickname_lower`，新注册用户的该字段默认为空字符串 `''`，第二个用户注册时与第一个冲突。

**修复**：`register.php` 第 69 行 INSERT 语句补上 `nickname_lower` 字段，值为 `mb_strtolower($nickname, 'UTF-8')`。

**相关背景**：昨天（2026-03-28）在 `database_upgrade_v3.sql` 中给 `users` 表新增了 `nickname_lower` 字段并加了唯一索引，但 `register.php` 的 INSERT 没有同步更新。
