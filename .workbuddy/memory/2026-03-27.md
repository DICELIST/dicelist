## TRPG Bot 导航网站开发完成

### 项目路径
`d:/桌面/代码开发/信息展示网站/`

### 项目简介
TRPG 线上跑团 Bot 工具展示平台，让骰主公开展示自己搭建的 Bot。

### 技术栈
- 后端：PHP 8.2（原生，无框架）
- 数据库：MySQL 5.7.44，数据库名 dicelist，用户名 dicelist
- 前端：HTML5 + CSS3 + 原生 JavaScript
- Web服务器：Nginx 1.24.0
- 管理工具：宝塔面板
- Markdown 渲染：marked.js + DOMPurify（CDN引入）

### 生成文件清单
- database.sql — 建表及初始数据
- includes/config.php — 数据库及站点配置
- includes/db.php — PDO 单例
- includes/functions.php — 公共函数（Session/CSRF/Flash/分页等）
- includes/header.php / footer.php — 公共模板
- assets/css/style.css — 全局样式（科技蓝白+背景网格线）
- assets/js/main.js — 汉堡菜单/Markdown预览/Flash自动关闭
- index.php — 首页（列表+搜索+筛选+排序+分页）
- detail.php — 详情页（Markdown渲染+浏览计数）
- submit.php — 提交页（登录后可用）
- edit.php — 编辑页（仅作者或管理员）
- delete.php — 删除（权限校验）
- login.php / register.php / logout.php — 用户系统
- profile.php — 个人中心（修改信息+我的Bot列表）
- admin/index.php — 仪表盘
- admin/users.php — 用户管理
- admin/bots.php — Bot管理
- admin/options.php — 下拉选项管理（软删除+排序）
- admin/settings.php — 网站设置（版权/备案/站名）
- nginx.conf — Nginx 配置参考
- DEPLOY.md — 完整部署教程

### 部署 Bug 修复记录

1. **数据库密码错误**：config.php 中密码与宝塔实际设置不符，需对照宝塔数据库页面确认。
2. **includes/header.php 路径重复**：第7行 `require_once __DIR__ . '/includes/functions.php'` 错误，因 `__DIR__` 本身已指向 `includes/`，应改为 `require_once __DIR__ . '/functions.php'`。已修复本地文件，需同步到服务器。
